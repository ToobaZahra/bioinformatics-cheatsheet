# Linux Commands I Use

## Navigation
- 'pwd' - print working directory (shows where I am working)
- 'ls-lh' - list files with sizes
- 'cd'- go up one folder

## File operations
- 'cp file.txt backup/' - copy file.txt into backup folder
- 'mv old.txt new.txt' - rename or move file

## My gotchas
- Forgot: 'cd' alone goes to home directory
- Forgot: 'ls-a' shows hidden files starting with '.'
